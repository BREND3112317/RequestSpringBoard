# RequestSpringBoard
## Install
```
npm install
npm start
```
## Use
POST / GET / PUT / DELETE http://localhost:3000/url
```
POST http://127.0.0.1:3000/xxx.xxx.xxx.xxx:xxxx/AGV/SendAgvCmd
Content-Type: application/json
{
    "Name": "ITRI_3-1",
    "Cmd": "500",
    "Param": []
}
```
